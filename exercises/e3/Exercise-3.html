Exercise 3: CSS Fonts & Animation

Assignment parameters: Display three different action verbs. Each word must be typeset with a custom font from Adobe Fonts, Google Fonts and lastly a self-hosted font, each word/typeface contain a fallback font stack. Using CSS, animate the action verbs you chose based on the action they describe, this animation must be triggered through hovering.

Assignment Objective: To gain familiarity with font services for the web, and learn how to implement them. This will help with your second and final project, Web Typographic Poster and Index assignments. In addition, thi will show how animation can be used to enhance meaning and demonstrate the first layer of interactivity.
