Exercise 5: Element Inspector

Assignment Parameters: Select a pre-existing new homepage from the following: the Canadian Broadcasting Corporation (CBC), The Globe and Mail, and/or The New York Times. Using the inspect tool, edit all of the content above the fold to make it about yourself.
Assignment Objective: To use the inspect tool as a means of investigating the structure of a website and to also show that information can be easily edited (at least locally) on any site.
