Exercise 2: CSS Flags

Assignment parameters: Draw 5 flags using only HTML and CSS, the five flags you are required to create are; Sweden, Denmark, Finland, Iceland and lastly your own original design flag.
Assignment Objective: To help you with your first project by familiarizing yourself with grids and using CSS as a medium for drawing.
